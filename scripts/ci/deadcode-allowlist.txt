# deadcode allowlist — known-acceptable unreachable functions.

# scripts/ci/deadcode.sh (and `make deadcode`) fails on any deadcode output
# line not listed here. Lines are compared with the :LINE:COL position

#   <file>: unreachable func: <func>

# explaining why the dead code is intentionally kept; stale entries (no
# longer reported) produce a warning so the list only ever shrinks.
