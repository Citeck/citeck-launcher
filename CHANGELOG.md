# Changelog

Release notes are maintained **per release, per language** in the
[`changelog/`](changelog/) directory — one folder per version (`changelog/<version>/`)
with a Markdown file per locale: `en.md`, `ru.md`, `zh.md`, `es.md`, `de.md`, `fr.md`,
`pt.md`, `ja.md`. English (`en.md`) is the canonical source and the fallback for any
missing locale.

- **Browse every version:** [`changelog/`](changelog/)
- **Downloads + notes per release:** [GitHub Releases](https://github.com/Citeck/citeck-launcher/releases)

The desktop app also shows these notes in-app when an update is available.
