## Исправления
- Маркеры изменений в редакторе конфигурации теперь работают для полей внутри списков (например, настроек проб), а неизменённые значения по умолчанию больше не помечаются как добавленные.
- Диалог снапшотов теперь поясняет, почему кнопки «Создать» и «Импорт» недоступны (сначала нужно остановить namespace).
- «Открыть каталог namespace» теперь открывает саму папку namespace, а не её внутренний подкаталог runtime-файлов.
- В Windows десктоп-приложение теперь корректно запускается при чистой установке.
