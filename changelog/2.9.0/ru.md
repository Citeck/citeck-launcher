## Новое
- **HTTPS для неймспейса (самоподписанный).** Новая опция «Включить HTTPS (самоподписанный)» в диалоге неймспейса отдаёт прокси по HTTPS с автоматически сгенерированным сертификатом. Порт следует за схемой автоматически — 443 для HTTPS, 80 для HTTP — поэтому включение и выключение HTTPS просто работает. При первом заходе браузер покажет предупреждение о самоподписанном сертификате.

## Изменения
- **Правки конфигурации точечные и мгновенные.** Редактирование настроек неймспейса или приложения теперь пересоздаёт только те контейнеры, чья конфигурация действительно изменилась, и показывает изменение сразу — образы непричастных сервисов больше не перекачиваются. Чтобы подтянуть свежие образы, используйте «Обновить и запустить».
- **Запуск ждёт мастер-пароль.** Если неймспейс тянет образы из приватного реестра, а секреты заблокированы, он теперь ждёт ввода мастер-пароля перед стартом — и диалог учётных данных больше не перекрывает экран разблокировки.
