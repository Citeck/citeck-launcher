## Fixes
- Drop-down menus in the create/edit namespace dialog (bundle repository, version, authentication type) could not be opened — fixed.
- Newly released platform versions appear in the version picker again: a stale local copy of the workspace no longer hides them, and Quick Start now shows the resolved version instead of "LATEST".
- The application menu icon no longer disappears after installing or updating on Linux.
- The theme and interface language are no longer reset after a desktop update.
- The desktop app writes its daemon log file again (it was empty in desktop mode), which is needed to diagnose problems.
