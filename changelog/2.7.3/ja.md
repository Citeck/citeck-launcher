## 新機能
- ようこそ画面から直接ワークスペース設定（workspace-v1.yml）を編集できます。ワークスペース選択の横の歯車で、変更マーカー付きの YAML エディターが開きます。変更は git リファレンスへの差分として保存され、同期のたびに再適用されます。
- スナップショット作成時に含めるボリュームを正確に選択できます。

## 修正
- デスクトップアプリのステータスがすべてのプラットフォームでリアルタイムに更新されるようになりました。Windows では、長い起動中に（例：すべて「読み込み中」のまま）「停止」を押すまで固まって見えることがなくなりました。
- Windows で、マスターパスワードの確認後やネストしたダイアログを閉じた後にウィンドウがクリックできなくなる問題を修正しました。
- イメージのプル失敗時に原因がログに記録されるようになり（例：レジストリ接続の拒否）、プルが通らなかった理由が分かります。
- ネームスペース一覧にローカライズされたステータスが表示され、削除されたネームスペースはクイック起動の一覧から消えます。
- ようこそ画面がライトテーマで読みやすくなり、ネイティブのチェックボックスが両テーマで正しく整列して表示されます。
- 名前変更後にネームスペースのヘッダーが確実に更新され、ドロップダウンで値を選んでもレイアウトがずれなくなりました。
- インストーラーが Windows のインストールまたはアップグレード後にアプリを自動的に起動するようになりました。
