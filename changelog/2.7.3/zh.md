## 新功能
- 可直接在欢迎屏幕编辑工作区配置（workspace-v1.yml）——工作区选择器旁的齿轮会打开带变更标记的 YAML 编辑器。您的修改会作为 git 基准之上的增量保存，并在每次同步时重新应用。
- 创建快照时可精确选择要包含的卷。

## 修复
- 桌面端状态现在在所有平台上实时更新；在 Windows 上，长时间启动期间状态不再卡住（例如全部显示“加载中”）直到您按下“停止”。
- 在 Windows 上，确认主密码或关闭嵌套对话框后，窗口不再变得无法点击。
- 镜像拉取失败现在会在日志中记录根本原因（例如注册表连接被拒绝），便于了解拉取为何未成功。
- 命名空间列表显示本地化的状态，已删除的命名空间会从快速启动列表中消失。
- 欢迎屏幕在浅色主题下清晰可读，原生复选框在两种主题下都能正确渲染并对齐。
- 重命名后命名空间标题可靠地更新，下拉框中选择值不再导致布局偏移。
- 安装程序现在会在 Windows 安装或升级后自动启动应用。
