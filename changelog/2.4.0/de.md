## Neue Funktionen
- Desktop-Anwendung — zurück in der 2.x-Reihe (zuvor nur in 1.x): ein einziges Fenster, um Citeck zu installieren und zu starten, mit nativen Installern für Windows, macOS und Linux. Installiert sich sauber über die 1.x-Desktop-App.
- Integriertes Auto-Update: Installiere die neueste Version direkt in der App, mit Änderungsprotokoll. Updates werden heruntergeladen, geprüft (SHA-256) und auf deinen Befehl angewendet; ein fehlgeschlagenes Update wird automatisch zurückgerollt.
- Mehrere Arbeitsbereiche in der Desktop-App: mehrere Setups nebeneinander, jedes mit seiner eigenen Bundle-Quelle.

## Verbesserungen
- Das Upgrade von der 1.x-Desktop-App benötigt kein Java mehr: deine Daten werden beim ersten Start migriert und vorher gesichert.
