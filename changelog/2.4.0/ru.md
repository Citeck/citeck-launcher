## Новые возможности
- Десктоп-приложение — вернулось в ветку 2.x (раньше было только в 1.x): одно окно, чтобы установить и запустить Citeck, с нативными установщиками для Windows, macOS и Linux. Ставится поверх десктоп-приложения 1.x без остатков.
- Авто-обновление приложения: установка последнего релиза прямо из приложения, со списком изменений. Обновления скачиваются, проверяются (SHA-256) и применяются по вашей команде; при сбое выполняется автоматический откат.
- Несколько рабочих пространств в десктоп-приложении: держите рядом несколько конфигураций, у каждой — свой источник бандлов.

## Улучшения
- Обновление с десктоп-приложения 1.x больше не требует Java: ваши данные переносятся при первом запуске, предварительно создаётся резервная копия.
