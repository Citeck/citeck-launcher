## Novidades
- Aplicativo de desktop — de volta na linha 2.x (antes só na 1.x): uma única janela para instalar e executar o Citeck, com instaladores nativos para Windows, macOS e Linux. Instala de forma limpa sobre o aplicativo de desktop 1.x.
- Atualização automática no aplicativo (por enquanto, apenas no Linux): instale a versão mais recente dentro do aplicativo, com registro de alterações. As atualizações são baixadas, verificadas (SHA-256) e aplicadas sob seu comando; se falharem, há reversão automática.
- Vários espaços de trabalho no aplicativo de desktop: mantenha várias configurações lado a lado, cada uma com sua própria fonte de bundles.

## Melhorias
- Atualizar a partir do aplicativo de desktop 1.x não requer mais Java: seus dados são migrados na primeira inicialização, com backup antes.
