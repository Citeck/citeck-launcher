## 新機能
- デスクトップアプリ — 2.x 系に復活しました（以前は 1.x のみ）：ひとつのウィンドウで Citeck をインストールして実行でき、Windows・macOS・Linux 向けのネイティブインストーラーを提供します。1.x のデスクトップアプリにそのまま上書きインストールできます。
- アプリ内自動更新（現時点では Linux のみ）：アプリ内から最新リリースをインストールでき、変更内容も表示します。更新はダウンロードして検証（SHA-256）し、あなたの操作で適用します。失敗時は自動的にロールバックします。
- デスクトップアプリで複数のワークスペースに対応：複数の構成を並べて保持でき、それぞれに独自のバンドルソースを設定できます。

## 改善
- 1.x デスクトップアプリからのアップグレードに Java は不要になりました：初回起動時にデータを移行し、事前にバックアップを作成します。
