## New features
- Desktop application — back in the 2.x line (it was 1.x-only): a single window to install and run Citeck, with native installers for Windows, macOS, and Linux. Installs cleanly over the 1.x desktop app.
- In-app auto-update (Linux for now): install the latest release from inside the app, with a changelog of what changed. Updates are downloaded, verified (SHA-256), and applied on your command; a failed update rolls back automatically.
- Multiple workspaces in the desktop app: keep several setups side by side, each with its own bundle source.

## Improvements
- Upgrading from the 1.x desktop app no longer needs Java: your data is migrated on first launch and backed up first.
