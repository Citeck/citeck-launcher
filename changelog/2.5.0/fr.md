## Améliorations
- Les espaces de noms de l'application de bureau sont désormais stockés dans la base de données, de façon fiable et cohérente.
- « Rechercher des mises à jour » est toujours disponible dans le menu de mise à jour de l'application de bureau.

## Corrections
- L'import et l'export d'instantanés fonctionnent désormais dans l'application de bureau : les données des volumes sont restaurées correctement.
- La suppression d'un espace de noms ne laisse plus d'entrée résiduelle.
- L'icône de l'application de bureau apparaît désormais dans le menu des applications après l'installation sous Linux.
- Les menus déroulants ne sont plus tronqués dans les boîtes de dialogue, les panneaux et l'assistant de configuration.
- L'exécutable Windows inclut désormais l'icône de l'application.
- La mise à niveau depuis le launcher 1.x est plus robuste : une migration de données échouée est désormais relancée proprement au lieu de laisser un état vide.
