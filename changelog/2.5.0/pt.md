## Melhorias
- Os namespaces do aplicativo de desktop agora são armazenados no banco de dados, de forma confiável e consistente.
- "Verificar atualizações" está sempre disponível no menu de atualização do aplicativo de desktop.

## Correções
- A importação e exportação de snapshots agora funcionam no aplicativo de desktop — os dados dos volumes são restaurados corretamente.
- Excluir um namespace não deixa mais uma entrada residual.
- O ícone do aplicativo de desktop agora aparece no menu de aplicativos após a instalação no Linux.
- Os menus suspensos não são mais cortados em diálogos, painéis e no assistente de configuração.
- O executável do Windows agora inclui o ícone do aplicativo.
- A atualização a partir do launcher 1.x está mais robusta: uma migração de dados malsucedida agora é repetida de forma limpa em vez de deixar um estado vazio.
