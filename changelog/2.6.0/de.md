## Neue Funktionen
- Wiederverwendbare Registry-Zugangsdaten. Ein gespeicherter Login für eine private Registry lässt sich jetzt über mehrere Namespaces und Workspaces hinweg nutzen, statt ihn für jeden einzeln neu einzugeben. Außerdem prüft der Launcher vor dem Start eines Namespace, dass für jede benötigte private Registry Zugangsdaten vorliegen — ein fehlender Login wird so sofort erkannt und blockiert nicht erst später einen Download.

## Änderungen
- Die Bundle-Auswahl hat keinen eigenen Eintrag „LATEST" mehr: Die neueste Version ist mit „(LATEST)" markiert, und ein Namespace wird stets an eine konkrete Version gebunden, sodass er nie von selbst zu einer neueren Version wechselt.
